# Project Structure #
## Backend ##
- The backend is a java application. Using Spring and Spring OpenAPI
- The source code is in the `src/main/java/com/pinewoods/score/tracker` directory.
- The main config is locatied at `java/com/pinewoods/score/tracker/config/SecurityConfig.java`
## Frontend ##
- The frontend is a React application using Vite.
- The source code is in the `frontend` directory.
